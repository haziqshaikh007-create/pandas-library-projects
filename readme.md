# Pandas Projects & Tutorials

This repository showcases multiple **Pandas-based data analysis projects**.  
Each project demonstrates core Pandas functions for **data cleaning, manipulation, and visualization**.

## 📂 Projects Included
1. **Student Dataset Analysis**
   - Functions: `read_csv`, `head`, `info`, `describe`, `groupby`, `sort_values`
   - Focus: Student demographics and performance insights

2. **Mobile Dataset Analysis**
   - Functions: `dropna`, `fillna`, `value_counts`, `pivot_table`, `loc/iloc`
   - Focus: Mobile usage patterns and sales trends

3. **Cricketers Stats Analysis**
   - Functions: `merge`, `concat`, `apply`, `mean`, `max`, `min`
   - Focus: Comparative statistics of cricket players
